# Smart-Pond
Monitoring aquaculture system using IoT

## Components
 - Mega 2560 Pro
 - SIM808 GSM Module
 - OLED Display
 - Temperature Sensor      : DS18B20
 - pH Sensor               : DFRobot pH Meter v1.1
 - Dissolved Oxygen Sensor : Gravity Analog Dissolved Oxygen Sensor

## Libraries
 - Arduino.h
 - SoftwareSerial.h
 - OneWire.h            
 - DallasTemperature.h 
 - U8glib.h
